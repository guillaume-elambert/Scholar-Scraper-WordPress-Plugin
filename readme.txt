Allows integration of Google Scholar API in tour Wordpress website.

== Description ==
Scholar Scraper is a simple plugin that allows you to integrate Google Scholar into your Wordpress website.
It has been designed to be as customizable as possible while remaining simple to use.
Do not hesitate to participate in the development of this plugin!

== Installation ==
1. Download the code.
2. In the administration panel of Wordpress, under 'Plugins' > 'Add new', find the button 'Upload Plugin' at the top of the window and click on it.
3. Import the file using the form displayed and submit it.
4. Activate the Scholar Scraper plugin (either via the page that should show up or in the WordPress 'Plugins' menu).
5. Go to 'YOUR_SITE/wp-admin/admin.php?page=scholar_scraper' and configure the plugin settings.
6. To display the posts on your website, create/edit a post/page and add the block named 'Scholar Scraper'.
